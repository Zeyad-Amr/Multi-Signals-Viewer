# Multi-Signals-Viewer
- Flutter Mobile app displays multi signals.
- Ploting X and R Charts.
- Alarming at abnormal samples

## Technologies & Tools
- Flutter
- Arduino
- Bluetooth Module
- Pulse Oximeter
